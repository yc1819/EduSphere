# EduSphere

EduSphere is a web-based educational management platform designed to streamline academic and placement-related activities for students and teachers.

## Features

### Student Portal
- Student Dashboard
- Attendance Management
- Assignments
- Examinations
- Results
- Fee Management
- Library Management
- Study Materials
- Calendar & Timetable
- Inbox
- Placement Dashboard

### Teacher Portal
- Teacher Dashboard
- Attendance Management
- Assignment Monitoring
- Study Material Management
- Inbox Communication

### Placement Module
- View Placement Opportunities
- Job Listings
- Placement Updates
- Backend API Integration

## Tech Stack

### Frontend
- React.js
- JavaScript
- Vite
- HTML5
- CSS3
- Tailwind CSS

### Backend
- Node.js
- Express.js

## Installation

### Clone Repository

```bash
git clone https://github.com/yc1819/EduSphere.git
```

## Running Backend

Navigate to backend folder:

```bash
cd backend
node server.js
```

Server starts on configured port.

## Running Frontend

Open a new terminal:

```bash
cd student-teacher-dashboard
npm install
npm run dev
```

## Notes

- Placement Dashboard uses Backend APIs.
- Other modules currently use Local Storage for data persistence.
- Project is under active development.
